==========
Docker笔记
==========

`Docker`_ 是一个用于开发、分发以及运行应用的开源平台。
*Docker* 将应用与基础设施(硬件)分离，以便软件快速分发。
借助 *Docker* ，你可以采用与管理应用相同的方式来管理基础设施。
*Docker* 提供了一系列快速分发、测试以及部署代码的方法，
显著减少从编写代码到生产级运行的时间。
